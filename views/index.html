TimeStamp API

  • API endpoint: GET https://timestamp-api-gj.glitch.me/api/timestamp/:date_string?
  • If :data_string? is valid the api returns JSON with both UNIX and UTC date forms.
  • If :data_string? is empty the api returns current date and time (GMT)
  • If :data_string? is invalid the api returns {error: "Invalid Date"}

Examples

https://timestamp-api-gj.glitch.me/api/timestamp/2019-12-25
https://timestamp-api-gj.glitch.me/api/timestamp/1577232000000
